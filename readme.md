# FlexTable

## npm

```bash
npm install tm-flextable -save
// or
yarn add tm-flextable
```

## Feature

1. 基本配置-columns、loading、data、sum columns:
     - [x] title
     - [x] key
     - [x] width
     - [x] render
     - [x] sortable
     - [x] fixed left
     - [ ] fixed right

2. 基本功能
    - [x] 正常展示
    - [x] 滚动（body）
    - [x] 左右滚动（wrapper）
    - [x] 汇总数据
    - [x] 自定义列宽
    - [x] 单元格宽度计算
    - [x] 拖动调整列宽（resizable）
    - [x] 行loading(row.loading)
    - [x] selectable
    - [ ] 拖动改变列顺序
    - [ ] 子表格
